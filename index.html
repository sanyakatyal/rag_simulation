Knowledge Base

RAG stands for Retrieval Augmented Generation and improves LLM accuracy.
Embeddings convert text into vectors for semantic similarity search.
Vector databases store embeddings and allow fast retrieval.
Chunking splits documents into smaller passages for better retrieval.
The retrieved chunks are added to the prompt before sending to the LLM.
